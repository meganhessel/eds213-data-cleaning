Clean Water_cover Column

Dealing with non-numeric values

Look at the non-numeric values

# Investigate the non numeric values 
snowsurvey_csv %>% 
  count(Water_cover) %>% 
  filter(is.na(as.numeric(Water_cover)))
# A tibble: 5 × 2
  Water_cover     n
  <chr>       <int>
1 -              10
2 .             575
3 n/a            32
4 unk             1
5 <NA>          149

There are characters: - . n/a unk

These characters do not tell us any information about the values. Therefore, we must change all the non-numeric values to NAs.

# Change all the non-numeric values 
snowsurvey_fixed <- snowsurvey_csv %>% 
  mutate(Water_cover = ifelse(Water_cover == "-", NA, Water_cover), 
         Water_cover = ifelse(Water_cover == ".", NA, Water_cover),
         Water_cover = ifelse(Water_cover == "n/a", NA, Water_cover),
         Water_cover = ifelse(Water_cover == "unk", NA, Water_cover)
         )

# Check if it worked
snowsurvey_fixed %>% 
  count(Water_cover) %>% 
  filter(is.na(as.numeric(Water_cover)))
# A tibble: 1 × 2
  Water_cover     n
  <chr>       <int>
1 <NA>          767

Now that we removed the characters, we can make the column numeric.

# Make the column numeric 
snowsurvey_fixed$Water_cover <- as.numeric(snowsurvey_fixed$Water_cover)

Clean Snow Cover column

We did this in class: fix non-numeric and impossible values - make NAs.

# Fix the non-numeric values 
snowsurvey_fixed <- snowsurvey_fixed %>%
  mutate(Snow_cover = ifelse(Snow_cover == ".", NA, Snow_cover), 
         Snow_cover = ifelse(Snow_cover == "-", NA, Snow_cover), 
         Snow_cover = ifelse(Snow_cover == "n/a", NA, Snow_cover), 
         Snow_cover = ifelse(Snow_cover == "unk", NA, Snow_cover), 
         Snow_cover = ifelse(Snow_cover == "<1", 0, Snow_cover))

# Make the column numeric 
snowsurvey_fixed$Snow_cover <- as.numeric(snowsurvey_fixed$Snow_cover)

# Change negative, impossible values to NA 
snowsurvey_fixed <- snowsurvey_fixed %>% 
  mutate(Snow_cover = ifelse(Snow_cover > 100, NA, Snow_cover))

Recompute the Total Cover

Ensure the 3 columns (snow, water, and land cover) sum to 100 for each row.

The metadata said these rows must sum to 100. Therefore, when they did not sum to 100, I removed the row.

# Recalculate the Total Cover = Sum of snow, water, and land cover
snowsurvey_fixed <- snowsurvey_fixed %>% 
  mutate(Total_cover = rowSums(pick(Snow_cover, Water_cover, Land_cover), na.rm = TRUE))

# Investigate where total_cover != 100
snowsurvey_fixed %>% 
  filter(Total_cover != 100) %>% 
  count()
# A tibble: 1 × 1
      n
  <int>
1  5383
# Total cover must = 100. Therefore, dropping all values not 100 
snowsurvey_fixed <- snowsurvey_fixed %>% 
  filter(Total_cover == 100)

# Make sure it worked 
snowsurvey_fixed %>% 
  filter(Total_cover != 100) %>% 
  count()
# A tibble: 1 × 1
      n
  <int>
1     0
